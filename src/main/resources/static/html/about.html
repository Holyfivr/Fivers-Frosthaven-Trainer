Hey there!

So you want to know more about this program?
This program is made to manage and edit the Frosthaven ruleset files more easily.

Currently it is limited to editing character values, but in future versions, you will be able to:
  • Add starting characters
  • Edit item costs and values
  • Edit ability card values
  • Edit enchant costs and values
