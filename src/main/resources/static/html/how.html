Fivers Frosthaven Trainer

Important note: You can not edit an ongoing campaign!
All edits will only apply to new campaigns started after saving the ruleset file.


Start by opening your Frosthaven ruleset file using the "Open Ruleset" option in the menu bar. Navigate to the folder where your Frosthaven ruleset files are stored, which should be in:
...\Steam\steamapps\common\Frosthaven\Frosthaven_Data\StreamingAssets\Rulebase

When you load the ruleset file for the first time, a backup of the original file will be created automatically. This backup can be restored at any time using the "File" > "Restore Original" option in the menu bar.
To create a manual backup of your current ruleset file, use the "File" > "Create Backup" option in the menu bar, select a name for the backup, and it will be saved in the same folder as the original ruleset file.

Don't forget to save your changes before closing the program, using the "File" > "Save Ruleset" option in the menu bar.
